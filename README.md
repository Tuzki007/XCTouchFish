# XCTouchFish
这是一个摸鱼的系列工具，包括基于Electron的PC版、Chrome的Plugin插件、VSCode插件、Jetbrains插件、iOS版、Android版。
主要的功能是帮助你上班摸鱼使用，可以通过调节窗口透明度，隐藏视频播放区域，底部有视频推荐位，点击即可播放，可以通过滑杆调节App的透明度
